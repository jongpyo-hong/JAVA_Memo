블록 유효 범위 : let과 const
var - 함수가 지역 범위, 함수가 아니면 변수의 값이 변경된다
	 - 변수의 중복 정의 가능 - 오류의 원인

let, const - 지역의 범위 -> { ... }
	- 변수의 중복 정의 불가
	- let 변수 선언자
	- const 상수 선언자
	
	변수 = 변하는 값
	상수 = 값이 변하지 않는 값
