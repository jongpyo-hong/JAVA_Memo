Array 생성자 객체(배열)
- 많은 데이터를 담기 위해서
- 많은 데이터를 쉽게 처리하기 위해서

1. 생성방법
new Array(값); -> 값 - 배열 공간의 갯수

new Array(값, 값, 값....); -> 값이 들어가 있는 배열 객체 생성

[값, 값 ....] = new Array() - 같은 의미

2. 배열 객체가 되는 조건
1) Array.prototype 상속받는다
2) 속성명 - 숫자
3) length 속성이 존재한다

객체의 속성명에 접근

참조변수.속성명(숫자명 앞에 X)
참조변수[속성명]

있는 속성(숫자)에 대입 - 변경
없는 속성(숫자)에 대입 - 추가
delete 참조변수[속성명(숫자)]

배열 - 추가, 수정, 삭제 전용 메서드가 정의되어 있다
(Array.prototype)

추가
	push(...items) : 맨끝에 추가
	unshift(...items) : 맨앞에 추가
	
삭제(꺼낸다)
	pop() : 끝의 요소를 제거하고 새로운 배열 생성
	shift() : 앞의 요소를 제거하고 새로운 배열 생성

중간의 요소를 삭제, 교체, 추가한다
	Array.splice(배열번호), (해당 배열번호 부터 지울갯수),(수정내용),(수정내용)....
	
	
객체 속성 값 : 원시타입 (숫자, 문자 ,논리값, undefined, null...)
				  객체 (+ 함수객체 ...)
				  
배열 객체의 값 - 원시타입 + 객체(모든 값)

prototype에 Symbol.iterator 속성이 있으면 -> for ... of 구문 사용가능
