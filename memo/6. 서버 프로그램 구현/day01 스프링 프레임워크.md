# 스프링 프레임워크

## 1. 스프링 프레임워크

### 프레임워크?
- 개발 표준화된 설계

- 의존 주입(Dependency inject : DI) 지원 - 객체의 통제를 위해 (전략패턴)
- AOP(Aspect-Oriented Programming) 지원
- MVC 웹 프레임 워크 제공
- JDBC, JPA 연동, 선언적 트랜잭션 처리 등 DB 연동 지원

- 스프링 데이터
- 스프링 시큐리티
- 스프링 배치

* @Configuration : 설정 클래스

* @Bean : 스프링에 관한 자바 객체
